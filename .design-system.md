# RepoMind Design System

## Brand Identity
**Product**: RepoMind - AI-powered code understanding platform  
**Style**: Professional SaaS with spacious, airy layout  
**Inspiration**: LeetCode-inspired clean, functional aesthetic

## Color Palette

### Primary
- **Purple 600**: `#9333ea` - Main brand color
- **Purple 700**: `#7e22ce` - Hover/active state
- **Purple 800**: `#6b21a8` - Darker accents

### Neutrals
- **White**: `#ffffff` - Primary background
- **Neutral 50**: `#fafafa` - Secondary background
- **Neutral 100**: `#f5f5f5` - Hover states
- **Neutral 900**: `#1a1a1a` - Text/dark elements

### Functional
- **Success**: `#10b981` (green)
- **Warning**: `#f59e0b` (amber)
- **Error**: `#ef4444` (red)
- **Info**: `#3b82f6` (blue)

## Typography
- **Font Family**: System fonts (-apple-system, Segoe UI, Roboto)
- **Headings**: Font weight 600, tight line-height (1.2)
- **Body**: Font weight 400, comfortable line-height (1.6)
- **Monospace**: Fira Code or Monaco for code blocks

## Layout Principles
1. **Spacious & Breathing**: Generous padding (2rem-4rem sections)
2. **Max-width**: 1280px for optimal readability
3. **Mobile-first**: Responsive from 375px+
4. **Clear Sections**: Well-defined page sections with clear hierarchy
5. **Whitespace**: Abundant use of whitespace for premium feel

## Component Styles

### Buttons
- **Primary**: Purple background (#9333ea), white text
- **Secondary**: Light gray background, neutral text
- **Outline**: Transparent with border, transitions to purple on hover

### Cards
- **Background**: White with subtle border (#e5e5e5)
- **Hover**: Light shadow and border color shift
- **Padding**: 2rem (32px) for spacious feel

### Navigation
- **Style**: Sticky header with white background and backdrop blur
- **Logo**: Gradient badge (purple) with initial "R"
- **Actions**: Purple CTA buttons, neutral links

## Sections (Landing Page Structure)
1. **Navigation** - Sticky header with brand and auth
2. **Hero** - Large headline, badge, CTAs, demo box
3. **How it Works** - 3-step numbered process
4. **Features** - 6-card grid (Chat, Interview, Debug, Analysis, Speed, Privacy)
5. **Use Cases** - 4-card grid (Onboarding, Interviews, Tech Debt, Debugging)
6. **Final CTA** - Gradient background section
7. **Footer** - Dark background with branding

## Accessibility
- Clear color contrast (WCAG AA standard)
- Focus states for keyboard navigation
- Semantic HTML structure
- Alt text for images/icons
- Responsive touch targets (min 44px)

## Design Philosophy
✓ Professional, not playful
✓ Developer-centric, not enterprise-heavy
✓ Clean and minimal, not cluttered
✓ Spacious and breathing, not dense
✓ Consistent branding throughout
✓ No generic AI-generated appearance
